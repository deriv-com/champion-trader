BuyContract purchases a new contract

Request Body schema: application/json
required
buyRequestId
string (Unique ID to identify the request)
instrumentId
string (The instrument to trade)
productType
string (Type of the trading product)
stake
string (Amount to stake as a decimal string for precise handling)
stakeCurrency
string (Currency code of the stake amount)
duration
string <int64> (Length of the contract duration)
durationUnits
string (Duration units for contracts)
Default: "DURATION_UNITS_UNSPECIFIED"
Enum: "DURATION_UNITS_UNSPECIFIED" "DURATION_UNITS_TICKS" "DURATION_UNITS_SECONDS" "DURATION_UNITS_DAYS"
  • DURATION_UNITS_UNSPECIFIED: Default value when duration unit is not specified
  • DURATION_UNITS_TICKS: Duration measured in number of price ticks
  • DURATION_UNITS_SECONDS: Duration measured in seconds
  • DURATION_UNITS_DAYS: Duration measured in days
proposalDetails
string <byte> (Additional product-specific configuration)
